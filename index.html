Bootstrap Alerts

Bootstrap provides an easy way to create predefined alert messages. They each have their corresponding colors, just as the tables above, and can have manny effects, links, they can have an option to close, and can even incorporate animation.

This will work by incorporating several classes related to the type of alert you're looking for. Below we can find examples which incorporate the different alert types.

Success! Indicates a successful or positive action. .alert .alert-success
× Info! Indicates a neutral informative change or action. .alert .alert-info. You can click on the X symbol to close it. This uses data-dismiss aria-label inside a link. You can get more info here
Warning! Indicates a warning that might need attention. .alert .alert-warning This adds a link using the alert-link class.
× Warning! Indicates a warning that might need attention. .alert .alert-danger. This is also a closable alert but this time with an animation. this uses the classes alert-dismissible and fade in.

Bootstrap Buttons

Button Styles

Bootstrap provides different styles of buttons, which can be used on links, buttons or input objects. They can be:

  • Different sizes
  • Block Level (or content-size)
  • Active/Disabled (or ontent-size)





Button Groups

Bootstrap allows you to group a series of buttons together (on a single line) in a button group. You should use the .btn-group class on the div surrounding the buttons. The buttons can also be resized as a group, applying the .btn-group-lg|sm|xs classes appropriately

Other classes which can be used in buttons groups are: .btn-group-vertical, to create a vertical button group; .btn-group-justified, to span your buttons the entire width of the screen; and you can also create dropdown menus with together or split buttons. The whole list is here

Glyphicons

Bootstrap includes 260 glyphs from the Glyphicon Halflings set. Glyphicons Halflings are normally not available for free, but their creator has made them available for Bootstrap free of cost. As a thank you, you should include a link back to Glyphicons whenever possible.

Progress Bars can be used to show a user how far along he/she is in a process. It's as easy as using the .progress class with the div element.

They can come with percentage labels, text, can be colorful (using, also, the contextual classes, they can be stripped or have an animation on the fill. They can also be stacked (by placing multiple bars into the same div). The main class after the main div would be .progress-bar but other values should be changed.
